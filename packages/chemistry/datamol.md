## üìñ Overview

Datamol is a lightweight, RDKit-based Python library that provides a simple, high-level API for molecular processing‚Äîcovering molecule I/O, conversion (e.g., SMILES/InChI/SELFIES), sanitization/standardization, fingerprinting, visualization, and parallelized workflows.
